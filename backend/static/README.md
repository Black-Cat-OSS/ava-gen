# Директория статического контента

Эта директория содержит статические файлы (HTML, CSS, JavaScript), которые раздаются через NestJS приложение.

## Структура

```
static/
├── index.html    # Главная страница
├── styles.css    # Файл стилей
├── script.js     # JavaScript файл
└── README.md     # Этот файл
```

## Доступ к файлам

Все файлы из этой директории доступны по URL с префиксом `/static/`:

- `http://localhost:3000/static/index.html`
- `http://localhost:3000/static/styles.css`
- `http://localhost:3000/static/script.js`

## Добавление новых файлов

1. Поместите файлы в эту директорию
2. Пересоберите проект: `npm run build`
3. Файлы будут автоматически скопированы в `dist/static/`

## Примечания

- Файлы автоматически копируются при сборке проекта
- В режиме разработки изменения отслеживаются автоматически
- Подробная документация: `backend/docs/STATIC_CONTENT.md`
